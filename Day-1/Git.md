
| Command           | Description                                                       | Example                                |
| ----------------- | ----------------------------------------------------------------- | -------------------------------------- |
| `git init`        | Initializes a new Git repository in the current directory         | `git init`                             |
| `git clone`       | Creates a local copy of a remote repository                       | `git clone https://repo-url.git`       |
| `git status`      | Shows the current state of the working directory and staging area | `git status`                           |
| `git add`         | Stages file(s) for the next commit                                | `git add file.txt`                     |
| `git add .`       | Stages all modified and new files                                 | `git add .`                            |
| `git commit`      | Saves staged changes with a message                               | `git commit -m "Initial commit"`       |
| `git log`         | Displays commit history                                           | `git log`                              |
| `git diff`        | Shows differences between changes                                 | `git diff`                             |
| `git branch`      | Lists, creates, or deletes branches                               | `git branch feature`                   |
| `git checkout`    | Switches to another branch                                        | `git checkout main`                    |
| `git checkout -b` | Creates and switches to a new branch                              | `git checkout -b feature`              |
| `git merge`       | Merges another branch into the current branch                     | `git merge feature`                    |
| `git remote`      | Shows remote repositories                                         | `git remote -v`                        |
| `git pull`        | Fetches and merges changes from a remote repo                     | `git pull origin main`                 |
| `git push`        | Uploads local commits to a remote repo                            | `git push origin main`                 |
| `git fetch`       | Downloads changes without merging                                 | `git fetch`                            |
| `git reset`       | Unstages or removes commits                                       | `git reset file.txt`                   |
| `git rm`          | Removes files from Git tracking                                   | `git rm file.txt`                      |
| `git stash`       | Temporarily saves uncommitted changes                             | `git stash`                            |
| `git stash pop`   | Restores stashed changes                                          | `git stash pop`                        |
| `git config`      | Sets Git configuration values                                     | `git config --global user.name "Name"` |
