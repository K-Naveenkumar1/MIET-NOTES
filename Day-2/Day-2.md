## UI/UX Principles

![Image](https://guides.uxtweak.com/wp-content/uploads/2023/02/UXT_guide_08.png)



### UI (User Interface)

Focuses on **visual design and interaction**.

- Buttons, icons, colors, typography
    
- Layout and spacing
    
- Visual consistency
    

### UX (User Experience)

Focuses on **how users feel** while using a product.

- Ease of use
    
- Efficiency
    
- Satisfaction
    

### Core UI/UX Principles

1. **User-Centered Design** – Design based on user needs
    
2. **Consistency** – Same patterns across the interface
    
3. **Visual Hierarchy** – Guide attention using size, color, contrast
    
4. **Feedback** – Show response to user actions
    
5. **Accessibility** – Design for all users (contrast, readable text)
    
6. **Simplicity** – Avoid unnecessary elements
    
7. **Usability** – Easy to learn and navigate

### What is Figma?

A **cloud-based UI/UX design tool** used for:

- Designing interfaces
    
- Prototyping
    
- Team collaboration
    

### Key Features

- **Frames** – Containers for screens
    
- **Shapes & Text** – Basic design elements
    
- **Components** – Reusable design elements
    
- **Auto Layout** – Responsive spacing and alignment
    
- **Styles** – Colors, text, effects
    
- **Prototyping** – Link screens with interactions
    
- **Real-time Collaboration**
    

### Advantages

- Works in browser
    
- No installation required
    
- Easy sharing and feedback

## HTML5 Semantics

![Image](https://media.geeksforgeeks.org/wp-content/uploads/20241127115416984900/html-sementics-layout.png)


### What are Semantic Elements?

Tags that **describe their meaning** clearly to browsers and developers.

### Common Semantic Tags

- `<header>` – Top section of page
    
- `<nav>` – Navigation links
    
- `<main>` – Main content
    
- `<section>` – Thematic grouping
    
- `<article>` – Independent content
    
- `<aside>` – Sidebar content
    
- `<footer>` – Bottom section
    

### Benefits

- Better SEO
    
- Improved accessibility
    
- Clean and readable code

## CSS Flexbox

![Image](https://css-tricks.com/wp-content/uploads/2018/11/00-basic-terminology.svg)



### Purpose

One-dimensional layout (row **or** column)

### Key Concepts

- **Flex Container**: `display: flex`
    
- **Main Axis & Cross Axis**
    

### Important Properties

- `flex-direction` – row / column
    
- `justify-content` – alignment on main axis
    
- `align-items` – alignment on cross axis
    
- `flex-wrap` – wrapping items
    
- `gap` – spacing between items
    

### Best Used For

- Navigation bars
    
- Buttons
    
- Cards in a row

## CSS Grid

![Image](https://ishadeed.com/assets/grid-area/grid-areas-0.png)

### Purpose

Two-dimensional layout (rows **and** columns)

### Key Concepts

- Grid Container: `display: grid`
    
- Rows and columns defined together
    

### Important Properties

- `grid-template-columns`
    
- `grid-template-rows`
    
- `gap`
    
- `grid-area`
    
- `grid-template-areas`
    

### Best Used For

- Page layouts
    
- Dashboards
    
- Complex designs
